\section{YAML Configuration Files Additional
Syntax}\label{yaml-configuration-files-additional-syntax}

All YAML 1.2.2 constructs are supported. Below are the specific
structures used in configuration files for defining operation
constraints, profiles, and approximation parameters.

\subsection{Key-Value Pairs}\label{key-value-pairs}

Key-value pairs are given as standard YAML mappings:\\
\texttt{key:\ \textless{}value\textgreater{}}.

\subsection{Lists}\label{lists}

Lists are given as standard YAML lists:\\
\texttt{{[}\ \textless{}item1\textgreater{},\ \textless{}item2\textgreater{},\ ...\ {]}}.

\subsection{Tuples}\label{tuples}

A tuple is given as a string bounded by parenthesis wrapped colons:\\

\begin{itemize}
\tightlist
\item
  \texttt{"(:\ \textless{}first\textgreater{},\ \textless{}second\textgreater{}\ :)"}.

  \begin{itemize}
  \tightlist
  \item
    In a list of tuples, all tuples must be of the same length.
  \end{itemize}
\end{itemize}

\subsection{Intervals}\label{intervals}

An interval is given as a string bounded by parenthesis wrapped bracket:

\begin{itemize}
\tightlist
\item
  \texttt{"({[}\ \textless{}lowerbound\textgreater{},\ \textless{}upperbound\textgreater{}\ {]})"}.

  \begin{itemize}
  \tightlist
  \item
    The brackets distinguish intervals from tuples.
  \end{itemize}
\end{itemize}

\subsection{Expressions}\label{expressions}

An expression is given as a string bounded by parenthesis wrapped
parenthesis:\\

\begin{itemize}
\item
  \texttt{"(\ \textless{}expression\textgreater{}\ )"}
\item
  a list is given:
  \texttt{"(\ {[}\ \textless{}item1\textgreater{},\ \textless{}item2\textgreater{},\ ...\ {]}\ )"}.
\item
  a tuple is given:
  \texttt{"(\ (:\ \textless{}first\textgreater{},\ \textless{}second\textgreater{},\ ...\ :)\ )"}.
\item
  an interval is given:
  \texttt{"(\ ({[}\ \textless{}lowerbound\textgreater{},\ \textless{}upperbound\textgreater{}\ {]})\ )"}.
\end{itemize}

\subsubsection{Iterators}\label{iterators}


In an expression, string interpolating iterators may be used to generate lists.

String interpolation is done with
\texttt{"prefix\$\{i\}suffix"}, where `i` is replaced:
\begin{verbatim}"( [ Binary4p{i}sf for i in [1..3] ] )"\end{verbatim}
$\mapsto$\begin{verbatim}[ Binary4p1sf, Binary4p2sf, Binary4p3sf ]\end{verbatim}

Multiple substitutions may be done within a single item
\begin{verbatim}"( [ (Binary4p$(i)s$(j) for i in [1..2] for j in ["f", "e"]) ] )"\end{verbatim}
$\mapsto$
\begin{verbatim}[ Binary4p1sf, Binary4p2sf, Binary4p1se, Binary4p2se ]\end{verbatim}

Two generated lists are concatenated when they are wrapped in parenthesis
\begin{verbatim}
"( ([ Binary6p$(i)sf for i in [1, 3, 4] ], [ Binary8p${k}se for k in [2..4] ]) )"
\end{verbatim}
$\mapsto$
\begin{verbatim}[ Binary6p1sf, Binary6p3sf, Binary6p4sf, Binary8p2se, Binary8p3se, Binary8p4se ]\end{verbatim}.
